# Learning Playwright Batch

This repository contains code samples and learning materials for Playwright automation testing.

## Project Structure

- **Chapter_01_Basics**: Basic concepts and setup verification.
- **chapter_02_Java_Concepts**: Java concepts relevant to the training (if applicable).

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/pritamnayak1980/LearningPlaywrightBatch.git
   ```
2. Navigate to the project directory:
   ```bash
   cd LearningPlaywrightBatch
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
   (Note: Ensure you have Node.js installed)

## Running Tests

To run the setup verification:
```bash
node Chapter_01_Basics/03_Verify_setup.js
```

## Prerequisites

- Node.js
- Git
